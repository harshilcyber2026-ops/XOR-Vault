Background

The binary stores the flag as an encrypted byte array in its .rodata section. On startup it XORs each byte with a key you supply and prints the result.


Single-byte XOR has only 256 possible keys. You can brute-force all of them in milliseconds — or use frequency analysis to guess the key from the cipher bytes alone.

Attack Approaches

A
Brute Force
Try keys 0x00–0xFF. Check which output is printable ASCII. Only 256 attempts.
B
Frequency Analysis
The most common cipher byte XORed with the most common plaintext char ('_' or 'e') approximates the key.
C
Known-Plaintext
Flags start with FlagVault{ — XOR first byte 0x1c with 'F' (0x46) gives key 0x5a instantly.
D
Static Analysis
Use strings, xxd, Ghidra, or radare2 to extract the encrypted buffer, then run approach A/B/C offline.

// Flags always start with "FlagVault{"

XOR is its own inverse: enc[0] XOR plain[0] = key

Python Solve Script

// solve.py
# Extracted from binary's .rodata section
enc = bytes([
    0x1c, 0x36, 0x3b, 0x3d, 0x0c, 0x3b, 0x2f, 0x36,
    0x2e, 0x21, 0x22, 0x6a, 0x28, 0x05, 0x31, 0x69,
    0x23, 0x05, 0x28, 0x69, 0x2c, 0x69, 0x28, 0x29,
    0x69, 0x3e, 0x05, 0x38, 0x23, 0x05, 0x69, 0x34,
    0x2e, 0x28, 0x6a, 0x2a, 0x23, 0x27
])

# Known-plaintext: first byte must be 'F' (0x46)
key = enc[0] ^ ord('F')
print(f"Key: 0x{key:02x}")

# Decrypt
flag = bytes([b ^ key for b in enc]).decode()
print(f"Flag: {flag}")
